Cluster Analysis¶

Cluster analysis offers another way for us to identify categories or topics. In most instances both LSA and LDA are considered "soft" clustering as each response can be partially assigned to each cluster whereas k-means clustering is considered "hard" clustering as each response is assigned to exactly 1 topic. For more specifics on k-means I'd point you to the following article.

The first thing we want to do is identify the # of clusters we want to use. We can do that using the elbow method and a for loop.

from sklearn.cluster import KMeans

wcss=[]

for i in range(3,12):
    kmeans = KMeans(n_clusters=i, init='k-means++', max_iter=300, n_init=10, random_state=0)
    kmeans.fit(vectors)
    wcss.append(kmeans.inertia_)
